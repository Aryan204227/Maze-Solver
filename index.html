How DFS & Backtracking Works

Depth-First Search (DFS) starts at the root node (Start Position) and explores as far as possible along each branch before backtracking.

When the algorithm runs into a wall or a corner where all neighbors are already visited (a dead end), it pops the current frame off the call stack, steps back to the previous coordinate, and checks the next open direction. This cycle repeats until the End Position is reached.
